---
返回将包含在指定最终交易区块中的已验证交易作为长度为 _i_ 的数组，其中 _i_ 是加上 DS 委员会的分片数。 交易根据处理交易的组进行分组。 数组的第一个元素指第一个分片。 索引处数组的最后一个元素 _i_ 指的是 DS 委员会处理的交易。

### HTTP 请求

| Chain(s)              | URL(s)                       |
| --------------------- | ---------------------------- |
| **Zilliqa 主网**   | https://api.zilliqa.com/     |
| **开发者测试网** | https://dev-api.zilliqa.com/ |
| **本地测试网**     | http://localhost:4201/       |
| **隔离服务器**   | https://zilliqa-isolated-server.zilliqa.com/ |

### 参数

| 参数 | 类型   | 是否必填 | 描述                                        |
| --------- | ------ | -------- | -------------------------------------------------- |
| `id`      | string | 必填 | `"1"`                                              |
| `jsonrpc` | string | 必填 | `"2.0"`                                            |
| `method`  | string | 必填 | `"GetTransactionsForTxBlock"`                      |
| `params`  | string | 必填 | 返回指定的 TX 区块高度。 如： `"2"` |
