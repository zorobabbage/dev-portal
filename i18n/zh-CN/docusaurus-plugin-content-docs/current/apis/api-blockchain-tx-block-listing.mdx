返回指定页面的最多 **10** 个交易区块及其区块哈希的分页列表。 还返回指定最大可用页面数的 `maxPages` 变量。

### HTTP 请求

| Chain(s)              | URL(s)                       |
| --------------------- | ---------------------------- |
| **Zilliqa 主网**   | https://api.zilliqa.com/     |
| **开发者测试网** | https://dev-api.zilliqa.com/ |
| **本地测试网**     | http://localhost:4201/       |
| **隔离服务器**   | https://zilliqa-isolated-server.zilliqa.com/ |

### 参数

| 参数 | 类型   | 是否必填 | 描述                                                |
| --------- | ------ | -------- | ---------------------------------------------------------- |
| `id`      | string | 必填 | `"1"`                                                      |
| `jsonrpc` | string | 必填 | `"2.0"`                                                    |
| `method`  | string | 必填 | `"TxBlockListing"`                                         |
| `params`  | number | 必填 | 返回 TX 区块列表的指定页面。 如：`1` |
