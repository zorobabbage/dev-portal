---
Returns the status (code) of all unconfirmed Transactions.

Please refer to [GetPendingTxn](api-transaction-get-pending-tx#api-availability) for details on API availability and status codes.

```java
public class App {
    public static void main(String[] args) throws IOException {
```

```go
func GetPendingTxns() {
```


#### Zilliqa V6.2.0 and before

```json
{
  "id": "1",
  "jsonrpc": "2.0",
  "result": {
    "Txns": [
      {
        "Status": 1,

```

### HTTP 请求

| Chain(s)              | URL(s)                       |
| --------------------- | ---------------------------- |
| **Zilliqa 主网**   | https://api.zilliqa.com/     |
| **开发者测试网** | https://dev-api.zilliqa.com/ |
| **本地测试网**     | http://localhost:4201/       |
| **隔离服务器**   | https://zilliqa-isolated-server.zilliqa.com/ |

### 参数

| 参数 | 类型   | 是否必填 | 描述        |
| --------- | ------ | -------- | ------------------ |
| `id`      | string | 必填 | `"1"`              |
| `jsonrpc` | string | 必填 | `"2.0"`            |
| `method`  | string | 必填 | `"GetPendingTxns"` |
| `params`  | None   |          |                    |
