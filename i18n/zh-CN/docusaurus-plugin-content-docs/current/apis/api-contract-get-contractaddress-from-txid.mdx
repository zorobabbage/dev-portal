---
返回一个 20 字节的智能合约地址。  表示为一个 `String`。

**注意：** 这仅适用于合约部署交易。

### HTTP 请求

| Chain(s)              | URL(s)                       |
| --------------------- | ---------------------------- |
| **Zilliqa 主网**   | https://api.zilliqa.com/     |
| **开发者测试网** | https://dev-api.zilliqa.com/ |
| **本地测试网**     | http://localhost:4201/       |
| **隔离服务器**   | https://zilliqa-isolated-server.zilliqa.com/ |

### 参数

| 参数 | 类型   | 是否必填 | 描述                                                                                                      |
| --------- | ------ | -------- | ---------------------------------------------------------------------------------------------------------------- |
| `id`      | string | 必填 | `"1"`                                                                                                            |
| `jsonrpc` | string | 必填 | `"2.0"`                                                                                                          |
| `method`  | string | 必填 | `"GetSmartContracts"`                                                                                            |
| `params`  | string | 必填 | 一个 32 字节的交易 ID。<br/>如： `"AAF3089596437A7C6984FA2627B6F38B5F5B80FAEAAC6993C2E82C6A8EE2615E"` |
