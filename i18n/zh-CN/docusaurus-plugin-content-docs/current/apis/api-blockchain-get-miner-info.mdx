---
返回指定 DS 区块的挖矿节点（即 DS 委员会成员和分片）。

**注意：**

1. 省略了 Zilliqa Research 拥有的节点。
2. `dscommittee` 没有 `size` 字段，因为 DS 委员会的大小对于给定的链是固定的。
3. 对于 Zilliqa 主网，此 API 仅从 DS 区块 5500 开始可用。

### HTTP 请求

| Chain(s)              | URL(s)                       |
| --------------------- | ---------------------------- |
| **Zilliqa 主网**   | https://api.zilliqa.com/     |
| **开发者测试网** | https://dev-api.zilliqa.com/ |
| **本地测试网**     | http://localhost:4201/       |
| **隔离服务器**   | https://zilliqa-isolated-server.zilliqa.com/ |

### 参数

| 参数 | 类型   | 是否必填 | 描述                        |
| --------- | ------ | -------- | ---------------------------------- |
| `id`      | string | 必填 | `"1"`                              |
| `jsonrpc` | string | 必填 | `"2.0"`                            |
| `method`  | string | 必填 | `"GetMinerInfo"`                   |
| `params`  | string | 必填 | DS 区块高度。 如： `"5500"` |
