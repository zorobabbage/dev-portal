---
返回指定网络的 `CHAIN_ID`。 表示为一个 `String`。

不同链的 `CHAIN_ID` 见下表：

| Chain(s)              | `CHAIN_ID` |
| --------------------- | ---------- |
| **Zilliqa 主网**   | `1`        |
| **开发者测试网** | `333`      |
| **隔离服务器**   | `222`      |


**注意：** 从 `2` 到 `9` 的 `CHAIN_ID` 作为保留，供 Zilliqa 核心使用。

### 响应示例


```json
{
  "id": "1",
  "jsonrpc": "2.0",
  "result": "1"
}
```

### HTTP 请求

| Chain(s)              | URL(s)                       |
| --------------------- | ---------------------------- |
| **Zilliqa 主网**   | https://api.zilliqa.com/     |
| **开发者测试网** | https://dev-api.zilliqa.com/ |
| **本地测试网**     | http://localhost:4201/       |
| **隔离服务器**   | https://zilliqa-isolated-server.zilliqa.com/ |

### 参数

| 参数 | 类型   | 是否必填 | 描述       |
| --------- | ------ | -------- | ----------------- |
| `id`      | string | 必填 | `"1"`             |
| `jsonrpc` | string | 必填 | `"2.0"`           |
| `method`  | string | 必填 | `"GetNetworkId"`  |
| `params`  | string | 必填 | 空字符串 `""` |
