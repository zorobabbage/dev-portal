### HTTP 请求

| Chain(s)              | URL(s)                       |
| --------------------- | ---------------------------- |
| **Zilliqa 主网**   | https://api.zilliqa.com/     |
| **开发者测试网** | https://dev-api.zilliqa.com/ |
| **本地测试网**     | http://localhost:4201/       |
| **隔离服务器**   | https://zilliqa-isolated-server.zilliqa.com/ |

### 参数

| 参数 | 类型   | 是否必填 | 描述                                                                                      |
| --------- | ------ | -------- | ------------------------------------------------------------------------------------------------ |
| `id`      | string | 必填 | `"1"`                                                                                            |
| `jsonrpc` | string | 必填 | `"2.0"`                                                                                          |
| `method`  | string | 必填 | `"GetSmartContractState"`                                                                        |
| `params`  | string | 必填 | 20 字节的智能合约地址。 如： `"fe001824823b12b58708bf24edd94d8b5e1cfcf7"` |
