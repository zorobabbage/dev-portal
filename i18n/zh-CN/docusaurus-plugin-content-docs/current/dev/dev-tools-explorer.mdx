## Devex
<img alt="Viewblock" src={useBaseUrl('img/dev/tools/devex.png')} />

Devex 是由 Zilliqa Research 维护的开发者浏览器。 Devex 直接集成到开发人员工具中，如 Ceres 和 Scilla IDE。Devex 允许你查看 Zilliqa 网络的区块、地址、交易、状态以及合约代码、变量和任何 transition 事件。

<img alt="Viewblock" src={useBaseUrl('img/dev/tools/devexNetworkChange.png')} />

Devex 可在此 [链接](https://devex.zilliqa.com/) 获得，默认情况下，devex 指向 Zilliqa 主网，但你可以使用右上角的下拉按钮更改网络。

## 区块视图
<img alt="Viewblock" src={useBaseUrl('img/dev/tools/viewblock.png')} />

ViewBlock 允许你查看区块、地址、交易、Zilliqa 网络的状态、合约代码和变量。

| 网络 | 链接 |
| ------- | ---- |
| 主网 | https://viewblock.io/zilliqa |
| 测试网 | https://viewblock.io/zilliqa?network=testnet |

例如，[这个](https://viewblock.io/zilliqa/tx/c4030c73d6dae558ff0c9d98237101e342888115f13219a00bb14a8ee46fa3be?network=testnet)
是指向 transition `getHello()` 交易的链接。

如果你创建了一个合法代币并有一个徽标，请转到 [cryptometa](https://github.com/Ashlar/cryptometa) 仓库并按照自述文件说明进行操作。

### Viewblock 产品

[ViewBlock API](https://viewblock.io/api) - ViewBlock 的 API 是开发人员可能需要为其应用程序使用的一些其他方法的宝贵资源（例如，检索由特定地址发送的交易）。

[ViewBlock Zilliqa 状态](https://viewblock.io/zilliqa/stats) - ViewBlock 的统计数据提供了有关 Zilliqa 网络的统计信息，例如随时间推移的难度、块数和地址增长。

[ViewBlock 仪表板](https://dash.viewblock.io/d/zilliqa) - 是一个高级仪表板，显示更多统计信息，允许你自定义时间范围
