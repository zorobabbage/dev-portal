## Nucleus 钱包上的测试网水龙头

<img alt="Nucleus Wallet Faucet" src={useBaseUrl('img/dev/getting-started/nucleus-faucet.png')} />


为了将合约部署到 Zilliqa 的测试网并发送交易，你的帐户中需要有测试网 $ZIL。

在 nucleus 钱包的水龙头网站上输入你的钱包地址，一旦交易确认你将收到300个测试网 $ZIL。

水龙头可在 [https://dev-wallet.zilliqa.com/faucet](https://dev-wallet.zilliqa.com/faucet) 访问

## 源代码

如果你有兴趣研究水龙头的实现，你可以访问 GitHub 仓库 [https://github.com/Zilliqa/nucleus-wallet](https://github.com/Zilliqa/nucleus-wallet)
