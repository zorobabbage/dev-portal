在本节中，我们将构建用于访问 RentOnZilliqa 的前端应用程序。

## 技术

前端是使用 [Create React App](https://github.com/facebook/create-react-app) 构建的，并启用了 [TypeScript](https://www.typescriptlang.org)。 我们依靠 [Tailwind CSS](https://tailwindcss.com) 为应用程序设置样式。 这些设置可在相应的文档中免费获得。

使用 [ZilPay](https://zilpay.io)，我们将前端元素连接到智能合约的 transition 和状态。

## 页面

前端将包含一个显示所有可用房源的主页。 每个房源的详细信息将显示在单独的房源页面上。

### 房源页面

此页面列出了所有已发布在平台上的房屋。 对于拥有房东帐户的用户，还会显示由他们管理的房源。

## 模态

大多数操作，包括帐户和房源创建、预订等，都可以通过模态访问。

### 帐户创建和 ZilPay

在 [房源页面](#listings-page)，用户可以创建一个帐户。 这是通过模态完成的，它提供了连接 ZilPay 和帐户创建表单的选项。
