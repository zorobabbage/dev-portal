RentOnZilliqa，Zilliqa 全栈应用程序，是 Zilliqa 区块链上的一个成熟的应用程序，用于列出和出租房屋。 在本指南中，我们将完成构建全栈应用程序的过程，从编写合约到构建前端应用程序。

## 技术

- Scilla 智能合约。
- [ZilPay](https://zilpay.io) 用于访问用户钱包并与 Zilliqa 区块链交互。
- [创建 React App](https://github.com/facebook/create-react-app) 用于部署应用程序的前端。
- [Tailwind CSS](https://tailwindcss.com) 用于样式化应用程序。

## 应用程序的简要细节

用户可以创建一个帐户作为租户或房东。

如果用户是房东，他们可以创建房屋清单，其中包含房屋的详细信息（例如，标题、地址、便利设施）和每晚租金。他们还可以查看所有房屋的列表以及他们托管的房屋。他们可以编辑房屋的详细信息并将其从列表中删除。

如果用户是租户，他们可以从可用列表中租一个房子。如果房屋已出租且仍在租金有效期内，则该房屋不可用。一旦用户通过以 ZIL 支付金额租用房屋，我们就会将该房屋标记为不可用。该金额存储在合约中。

房东还可以查看其账户中累积的租金。他们可以随时提取租金。
