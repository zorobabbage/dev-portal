Rosetta 是 [Coinbase](https://blog.coinbase.com/introducing-rosetta-build-once-integrate-your-blockchain-everywhere-9b97d284f5b9) 的一个开源项目。如今，软件开发人员面临的主要问题之一是具有不同节点和钱包 API 的区块链数量不断增加。 Rosetta 的目标是将如何与区块链交互 [标准化](https://github.com/coinbase/rosetta-specifications)，让任何人都可以轻松地在区块链之上构建各种不同的用例。

对于新区块链项目的开发人员，Rosetta 接口可以更轻松地确保与使用 Rosetta 的交易所的兼容性，并且可以通过确保满足特定的安全条件来显著加快交易所与新区块链集成的时间，同时保护客户资金。

对于更广泛的加密开发者社区，Rosetta 使构建跨区块链应用程序（如区块浏览器、钱包和 dapps）变得更加容易。应用程序可以使用区块链项目的 Rosetta 实现来读取链上数据并以标准格式构建交易，而不是为每个支持的区块链编写自定义解析；最小化代码并简化维护。

# Zilliqa Rosetta

Zilliqa Rosetta 是 Zilliqa 区块链平台 Rosetta 规范的参考实现。数据和构造 API 已经实现。

最新版本的 Zilliqa Rosetta 可以在 https://github.com/Zilliqa/zilliqa-rosetta 找到

## 使用 Zilliqa 种子节点运行 Zilliqa Rosetta

要运行 Zilliqa Rosetta，请参阅 ["使用种子节点设置 Zilliqa Rosetta"](rosetta-setting-up-seed-node) 部分，了解如何设置它。你可以选择与 Rosetta 一起运行种子节点，或将 Rosetta 连接到我们的公共端点 `api.zilliqa.com`。

:::info
要运行种子节点选项，将需要白名单，白名单仅根据具体情况给出。
:::

## 在没有 Zilliqa 种子节点的情况下运行 Zilliqa Rosetta

要运行连接到 Zilliqa 公共 API 端点的 Zilliqa Rosetta，请参阅 [“设置 Zilliqa Rosetta 连接到公共 API 端点”](rosetta-setting-up-no-seed-node) 部分。

## 与 Zilliqa Rosetta 交互

你可以参考 Zilliqa Rosetta API 以获取支持的 Restful API 列表。如果需要 SDK，Rosetta 项目提供了 [Golang Rosetta SDK](https://github.com/coinbase/rosetta-sdk-go)。
